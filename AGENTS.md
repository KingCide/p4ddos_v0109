# 参考资料

`p4ddos_v0109/关键资料` 这个目录下，保存着本项目的设计文档，以及参考的论文。代码的设计思路，以及所需要的实验结果，都应该从这里寻找，都需要完全符合项目资料里的设计。

# 项目进度

本项目开发过程中的中间结果，中间状态，目前的进度，都应该放到`p4ddos_v0109/progress`这个目录下。

# 项目更新

每次项目更新的时候，都需要进行commit
1. 要把一次很多的更新拆成比较独立的commit，不要把很多更新放到一个commit
2. commit的描述信息要使用中文
